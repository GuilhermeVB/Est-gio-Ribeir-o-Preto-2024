# 2ª Ida - Situação A

# 2ª Ida - Situação B

# 2ª possibilidade (50% de chance) escolher a sala com a luz acesa.
- Em caso de falha, o interruptor da última sala seria descoberto, mas as outras não. (FRACASSO)
